## Single Purpose

OnlyCCFA has one purpose: to display CCF recommended ranks on academic search result pages and to filter Google Scholar results by CCF rank, helping users identify high-ranked conference and journal papers more efficiently.

## Host Permission Justification

OnlyCCFA needs access to supported academic search result pages to read visible paper titles and venue text, insert CCF rank badges, and provide the Google Scholar CCF rank filter. The extension only runs on the listed academic websites and does not access unrelated pages.

OnlyCCFA uses dblp.org only to query public DBLP publication metadata when a supported academic search page does not provide enough venue information locally. The extension uses this metadata to determine and display the CCF rank of the visible search result. It does not collect, sell, or upload user browsing data to the developer's server.

## Remote Code Use

OnlyCCFA does not execute remotely hosted code. All JavaScript, CSS, and extension logic are bundled inside the extension package. The extension may request public DBLP metadata as data, but it does not load or execute remote JavaScript, WASM, or other remote code.
